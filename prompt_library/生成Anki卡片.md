# 执行流程

1. **分析与搜索**：根据【主题】，结合当前年份（例如 2024-2025），构建精准的搜索查询（如 LLM Agent 最新面试题 或 RAG 技术 2024 进展），以找到相关的技术博客、论文、社区讨论或行业报告。
    
2. **浏览与学习**：分析搜索结果，选择2-3个最权威、信息最新的链接进行浏览和学习，深入理解该主题下的核心概念、最新进展和潜在挑战。
    
3. **综合与生成**：**严格基于你浏览和学习到的最新信息**，设计出高质量的面试题和答案。确保问题能够考察候选人对技术细节和前沿动态的理解。
    

# 输出格式与内容要求

你必须严格遵循以下所有指令：

## 格式：CSV 兼容

1. **分隔符**: 必须使用**分号 (;)** 作为字段分隔符。
    
2. **结构**: 每一行代表一张卡片，格式为 "正面问题";"背面答案"。
    
3. **无标题行**: 绝对不要生成任何标题行（如 "Front;Back"）。
    
4. **引号包裹**: 每一张卡片的正面和背面内容都必须用英文双引号 " " 包裹起来。
    

## 内容：HTML 格式化

- **答案部分（背面）** 必须使用HTML标签来格式化，以便在Anki中正确显示。
    
- 使用 <ul> 和 <li> 标签来创建项目符号列表。
    
- 使用 <b> 或 <strong> 标签来**加粗**核心关键词和概念。
    
- 答案应**简洁、精准、层次分明**，突出技术的重点。
